# 🤖 MTaiChatbot v3.1

A modern, full-stack AI assistant developed by **Masud Talukdar**. Built to showcase the power of **Gemini 3.1 Flash-Lite** with real-time web intelligence.

## ✨ Highlights
- **Real-Time Knowledge**: Uses Google Search Grounding to provide accurate info for **2026 events**.
- **Fluid UI**: Ultra-fast streaming responses with a "typing" effect.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile displays.
- **Developer-First**: Designed with a clean dark-mode aesthetic and Markdown support.

## 🛠️ Built With
- **Frontend**: React + Tailwind CSS + javascript
- **Backend**: Node.js + Express
- **AI Brain**: Gemini 3.1 SDK (March 2026 Release)

## 🚀 Quick Start
1. Add your `GEMINI_API_KEY` to a `.env` file in the backend.
2. Run `npm install` in both folders.
3. Start the server with `node server.js` and the frontend with `npm run dev`.
